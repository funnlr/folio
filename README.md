# FileShare — Preview & Share

Upload `.md`, `.pdf`, `.docx`, or `.txt` files, preview them instantly, and share to WhatsApp, Instagram, TikTok, X, Email, or Notes.

## Setup

```bash
npm install
npm run dev
```

## Deploy to Vercel

```bash
npm install -g vercel
vercel
```

Or connect the GitHub repo in the [Vercel dashboard](https://vercel.com).

## Built with
- React + Vite
- mammoth.js (DOCX parsing)
- react-markdown (Markdown rendering)
